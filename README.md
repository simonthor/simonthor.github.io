# simonthor.github.io
My personal website, containing information about me, my research, and my projects. There are also some fun side-projects that you can check out, such as a [fractal explorer](https://simonthor.github.io/fractals).

## Branches
The main branch is `react`, which contains the React + TypeScript source code for the website before compilation into Javascript. The code is automatically compiled and deployed to [simonthor.github.io](https://simonthor.github.io) using Github Actions whenever the `react` branch is updated.
